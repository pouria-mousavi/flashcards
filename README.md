# Flashcards PWA

A beautiful, dark-mode Flashcard app with Spaced Repetition (SM-2).

## Features
- **Smart Scheduling**: Anki-style algorithm (SM-2) with Learning steps.
- **Voice/TTS**: High-quality Text-to-Speech with voice selection.
- **Responsive Design**: "Noji-style" dark UI.
- **PWA**: Installable on Android/iOS.
- **Offline Capable**: Works without internet.
